# ePortfolio Website Template

This is a simple static ePortfolio site designed for your career task submission.

## Edit your content

Update these files:

- `index.html`
  - Replace `Your Name`, email, and LinkedIn URL
  - Rewrite the "About Me" text in your own words
  - Replace both artefact placeholder links with real file/URL links
- `styles.css`
  - Optional visual customization

## Run locally

Open `index.html` in your browser directly, or use VS Code/Cursor Live Server if available.

## Publish with GitHub Pages

1. Create a new GitHub repository (for example `eportfolio`).
2. Upload all files in this folder (`index.html`, `styles.css`, `script.js`, `README.md`).
3. In GitHub repository settings, open **Pages**.
4. Under **Build and deployment**, set:
   - Source: **Deploy from a branch**
   - Branch: **main** and folder **/(root)**
5. Save, then wait about 1-3 minutes.
6. Your website will be available at:
   - `https://<your-username>.github.io/<repo-name>/`

Use that URL as the link for Part B in your submission PDF.
